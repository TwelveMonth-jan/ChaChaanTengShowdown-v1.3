# 茶餐廳風雲

> 一款以香港茶餐廳午市高峰為主題的高互動桌遊。

午餐尖峰時段，客人源源不絕、要求千奇百怪。  
你不只要帶位、執單、上餐，還要防同事暗中加料、推客埋身、踢爆你蓋牌吹牛。

在《茶餐廳風雲》中，玩家扮演茶餐廳伙計，在混亂與壓力中求生。  
你要在有限桌位、有限手牌和逐步累積的「老闆怒氣值」之間取得平衡，撐到最後，成為全場唯一未被炒魷魚的人。

## 遊戲資訊

- 玩家人數：`3–5`
- 遊戲時間：`20–30 分鐘`
- 建議年齡：`8+`
- 版本：`V1.3`
- 類型：`高互動` `派對卡牌` `生存淘汰` `吹牛 / 陷害`

## 遊戲一句話

《茶餐廳風雲》是一款把茶餐廳文化、午市混亂、蓋牌吹牛和互相陷害混在一起的生存型卡牌桌遊。

如果你喜歡：

- 高互動桌遊
- 嘴砲和心理戰
- 「我明明快贏，點解俾人塞咗個奧客過嚟」這種戲劇性時刻
- 有主題感、有節奏、規則不算重但局勢很亂的派對遊戲

這款遊戲大概會很對味。

## 核心玩法

每位玩家面前都有自己的桌位、手牌和怒氣值。

你的目標是：

1. 把客人安排到自己的桌位
2. 收集正確食材為客人上餐
3. 賺取小費
4. 避免空桌、奧客和失誤讓老闆怒氣爆表

但問題是，其他玩家會一直搞你：

- 把麻煩客人硬塞到你桌上
- 在你上餐時偷偷「暗中加料」
- 等你蓋牌吹牛時當場踢爆
- 把快完成的客人跟你交換

當你的怒氣值達到角色上限，你就會被老闆即時開除。最後留在場上的玩家獲勝。

如果中央牌庫先抽完，遊戲進入「午市大清算」：

- 怒氣值最低者勝
- 若平手，小費最多者勝
- 若再平手，則共享勝利

## 一局遊戲的感覺

這不是一款安靜算分的遊戲。它比較像：

- 一邊經營自己桌面
- 一邊觀察其他人快不快崩潰
- 一邊決定什麼時候說謊、什麼時候踢爆
- 一邊把最麻煩的客人推給別人

遊戲重點不是單純「湊資源」，而是混亂中的判斷：

- 這一餐現在上安全嗎？
- 這張蓋牌會不會被抓？
- 這個奧客我要硬吃，還是反檯止損？
- 這回合要先保命，還是先害人？

## 特色亮點

### 1. 茶餐廳主題很直接

客人、奧客、食材、垃圾、功能牌全部都來自茶餐廳語境：

- 奶茶、菠蘿油、蛋撻、叉燒、公仔麵、煎蛋
- 上班族、學生哥、阿伯、護士姐姐
- 食評家、KOL、投訴王、黑面經理
- 蟑螂腳、洗潔精、煙頭、隔夜飯
- 強行帶位、暗中加料、老闆唔喺度、執枱神功、客滿

它不是把主題貼在規則外皮上，而是把主題直接變成互動來源。

### 2. 吹牛不是額外小機制，而是整局核心壓力

玩家可以在上餐時蓋牌，把手牌面朝下聲稱成自己需要的食材。  
其他玩家則可以在結算前喊出「你跣我！」進行踢爆。

這讓每次上餐都不只是資源湊齊沒，而是：

- 有沒有真的湊齊
- 有沒有在混
- 別人敢不敢抓
- 你敢不敢抓別人

### 3. 奧客不是高分卡，而是節奏炸彈

奧客需要 3 種食材，回報更高，但如果你沒及時處理，回合結束還會持續加怒氣。  
它迫使玩家在風險與收益間做選擇，也讓局勢自然變得很戲劇化。

### 4. 角色讓打法真正分化

V1.3 加入 6 位角色，每位角色都有不同的：

- 抗壓值
- 手牌上限
- 桌位數量
- 被動能力
- 主動技能

例如：

- `阿志` 比較靈活，手牌多，失誤容錯高
- `發哥` 更擅長踢爆與心理戰
- `肥威` 桌位少，但能把垃圾當食材用
- `小強` 桌位多，但抗壓低，節奏很拼

角色讓玩家開局前就能選風格，而不是每個人都只是在做同一件事。

## 回合流程

每位玩家的回合大致如下：

1. 抽牌  
   從中央牌庫抽 2 張牌。

2. 行動階段  
   你可以反覆做以下事情，直到你想停：
   - 帶位入座
   - 湊套上餐
   - 使用功能牌
   - 主動反檯
   - 花小費減怒氣
   - 發動角色主動技能

3. 上餐干擾與踢爆  
   當有人宣告上餐時，其他玩家可以暗中加料；如果有蓋牌，就能發起踢爆。

4. 查勤  
   回合結束時，空桌會增加怒氣。

5. 奧客結算  
   尚未處理的奧客會繼續施壓，追加怒氣。

## 勝利條件

你有兩種方式贏：

### 生存勝利

其他玩家全部被老闆炒掉，只剩你一個還在場上。

### 大清算勝利

當中央牌庫抽乾，遊戲立刻停止，進行最終結算：

- 怒氣最低者勝
- 平手看小費

## 設計方向

《茶餐廳風雲》的目標不是做成很重的策略桌遊，而是做成：

- 易理解但互動很強
- 局勢會一直變
- 有嘴砲空間
- 有主題記憶點
- 有明顯「香港茶餐廳」味道

整體定位偏向輕中度派對卡牌遊戲，但又保留足夠的資源管理、風險控制和角色差異，讓它不只是單純亂鬥。
